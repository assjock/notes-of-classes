### comparison operators

| | |
| --- | --- |
| x<y | is x less than y? |
| x<=y | is x less than or equal to y? |
| x==y | is x equal to y? |
| x!=y | is x not equal to y? |

two way decisions:

```python
x=10
if x>5:
    print('larger')
else: 
    print('smaller')
```

multi-way decision:
```python
x=10
if x>2:
    print('larger')
elif x>15:
    print('smaller')
else: #or no else
    print('medium')
```
